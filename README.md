# Literary Quiz App

A Django web application that tests users' knowledge of famous literary opening lines.

## Features
- Interactive quiz with classic book opening lines
- Instant feedback on answers
- Clean, responsive design

## Technologies Used
- Python
- Django
- JavaScript
- HTML/CSS

## How to Run
1. Clone the repository
2. Install dependencies: `pip install django`
3. Run server: `python manage.py runserver`
4. Visit `http://127.0.0.1:8000/`
